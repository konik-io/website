ZUGFeRD für Microsoft Dynamics NAV

ZUGFeRD Erweiterung mit nahtloser Integration für NAV Navision 2009-2018, Dynamics 365 und AX

Präsentation vereinbaren

Funktionsumfang ZUGFeRD für Microsoft Dynamics NAV

Rechnungsausgang

ZUGFeRD für NAV ermöglicht es, aus einer gebuchten Rechnung ZUGFeRD-konforme Rechnungen zu erzeugen und per E-Mail zu versenden. Durch einen Klick auf den 'ZUGFeRD Button' wird im Hintergrund die ZUGFeRD-konforme Rechnung erstellt und an den Standard-E-Mail-Client übergeben. In der neu erstellten E-Mail wird die ZUGFeRD Rechnung mit einem Text an den Empfänger als Anhang hinzugefügt. Die verwendete E-Mail Textvorlage lässt sich zudem für jeden Nutzer in NAV individualisieren.

Integration

Die gesamte Verarbeitung von ZUGFeRD und PDF ist in .NET Bibliotheken gekapselt. Dadurch wird die Abhängigkeit zu Navision interna auf ein Minimum reduziert. Für Sie bedeutet das zum Einen eine einfache und schnelle Integration und zum Anderen keinen Update Lockin an eine ganz bestimmte NAV Version.

Rechnungseingang

ZUGFeRD Eingangsrechnungen werden einer automatischen Rechnungsprüfung und -freigabe unterzogen, welche eine Dunkelverbuchung ohne menschliche Eingriffe ermöglicht. Je nach Vorfall kann die Eingangsrechnung einem Lieferanten und einer Bestellung zugeordnet werden oder bei einem Sofortkauf kann im Hintergrund eine Bestellung oder ein neuer Lieferant angelegt werden.
