Konik Bibliothek in Close Source Projekten

Die Konik Bibliothek für Java und .NET ist frei Verfügbar unter der AGPL Lizenz. Für Sie als Lizenznehmer und nutzer der Konik Bibliothek bedeuted dies das die entstandenen Werke ebenfalls unter AGPL Lizenz stehen.
Wir möchten das Sie die Konik Bibliothek auch in Werken verwenden welche nicht unter eine AGPL Lizenz gestellt werden können oder weil dadurch die Rechte dritter verletze werden können.

Aus diesem Grund bieten wir die Möglichkeit, die Konik Bibliothek ink. der PDF-Carriages auch unter einer Closed Source Lizenz zu erwerben.

Lizenzmodelle

Unternehmenslizenz Eine Unternehmenslizenz ermöglicht den einsatz der Konik Bibliothek inerhalb Ihres Unternehmens zu frei zu verwenden. Dies beinhaltet beliebieg viele Server, Clients und Mobile Anwendungen.

Serverlizenz Die Lizenz für die Konik Bibliothek auf einem Server oder in einer VM.

Desktop & Client Für Applikationen mit einem Userinterface die auf einem Desktop oder Mobilen Gerät laufen.

ERP Plugin Lizenz Konik ZUGFeRD Module und Plugins für ERP Systeme wie z.b. Microsoft Dynamics NAV (Navision), SAGE oder andere ERP Software auf Anfrage.

OEM Lizenzen Für ERP und DMS Sofwareanbieter bieten wir atraktive Lizenzmodelle der Konik Bibliothek.

Support Unsere Supportverträge lassen Sie nachts in ruhe schlafen. Wir helfen Ihnen auch bei Problemen mit interportabilität.
