Konik Bibliothek in Closed Source Projekten

Die Konik Bibliothek für Java und .NET ist frei verfügbar unter der AGPL Lizenz. Für Sie als Lizenznehmer und -nutzer der Konik Bibliothek bedeutet dies, dass die entstandenen Werke (Software/Online Dienste im Internet oder Intranet) ebenfalls unter der AGPL Lizenz stehen. Wir möchten, dass Sie die Konik Bibliothek auch in Werken verwenden, welche nicht unter eine AGPL Lizenz gestellt werden können oder weil dadurch die Rechte Dritter verletzt werden können.

Aus diesem Grund bieten wir die Möglichkeit an, die Konik Bibliothek inklusive der PDF-Carriages auch unter einer Closed Source-Lizenz zu erwerben.

Lizenzmodelle

Firmenlizenz Eine Unternehmenslizenz ermöglicht den freien Einsatz der Konik Bibliothek innerhalb Ihres Unternehmens. Dies beinhaltet beliebig viele Server, Clients und mobile Geräte.
Serverlizenz Die Lizenz für die Konik Bibliothek auf einem Server oder in einer VM.
Desktop & Client Für Applikationen mit einem Userinterface die auf einem Desktop oder mobilen Gerät laufen.
ERP Plugin Lizenz Konik ZUGFeRD Module und Plugins für ERP Systeme wie z.B. Microsoft Dynamics NAV (Navision), SAGE oder andere ERP Software auf Anfrage.
OEM Lizenzen Für Softwareanbieter die Konik in ihre Produkte integrieren möchten, bieten wir attraktive Lizenzmodelle.
Support Mit dem Supportvertrag profitieren Sie von garantierten Reaktions- und Problembehebungsfristen, Software Hotfixes und umfassender Beratung bei Integration oder ZUGFeRD Interoperabilitätsproblemen.
